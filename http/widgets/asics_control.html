This dialog represents the boards in your machine, and the ASICs in each board. In general, ASICs should be enabled. Disabling ASICs will degrade your mining capacity.
You can change the status of an ASIC by clicking on the relevant box.
Changes will be saved when you click "Apply".

Disabled state can be used to try to isolate noisy ASICs.
Removed status should only be set when an ASIC has been physically removed by a technician. This setting should not be selected by an end user.
